What I Learned:

Filtering Logic in React
  1. Store your main array (in my case, todos[]) in state.
  2. Create a filter state (filter) to track the current view, e.g. 'all' | 'active' | 'completed'.
  3. Use that state to render only the relevant todos:
const displayTodos = todos.filter(todo => {
if (filter === 'active') return !todo.complete;
if (filter === 'completed') return todo.complete;
return true;
});

It sounds simple now, but it took a few tries before it really clicked. Definitely planning to practice this more.
